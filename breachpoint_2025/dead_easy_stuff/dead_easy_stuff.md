# Dead Easy Stuff – DES Weak Key Challenge

**Event:** BreachPoint CTF  
**Category:** Cryptography  
**Difficulty:** Easy  

---

## Solution

### Challenge Setup

The ciphertext was generated by encrypting a plaintext once using **DES** with a **weak key**.

Example weak key used:


0x0101010101010101


Certain DES weak keys have a special property:


E_k(E_k(P)) = P


This means that **encrypting the ciphertext again with the same weak key returns the original plaintext**.

So instead of decrypting, we simply **encrypt again**.

### 2. Recognizing DES Weak Keys

DES has a **small set of weak keys**.

These keys produce special behavior where:


Encryption = Decryption


Some known weak DES keys include:


0x0101010101010101
0xFEFEFEFEFEFEFEFE
0xE0E0E0E0F1F1F1F1
0x1F1F1F1F0E0E0E0E


With these keys:


E_k(E_k(P)) = P


---

### 3. Testing the Weak Key

Instead of decrypting, we simply **encrypt the ciphertext again using the same weak key**.

Example using OpenSSL:

```bash
openssl enc -des-ecb -K 0101010101010101 -in ciphertext.bin -out output.txt
4. Recovering the Plaintext

Because the DES key is weak, the second encryption reveals the original plaintext:

PT = E_k(ciphertext)

Opening the output file reveals the flag.

Flag
BreachPoint{Told_you_it_was_DEAD_EASY}
Key Concept

The challenge demonstrates the dangers of weak cryptographic keys.

DES contains several weak keys that break normal encryption assumptions.

For these keys:

E_k(E_k(P)) = P

This property makes encryption self-inverting, allowing attackers to recover plaintext without performing decryption.

Lessons Learned
1. Weak Keys in Cryptography

Some cryptographic algorithms contain weak keys that should never be used.

DES has:

4 weak keys

12 semi-weak keys

2. Importance of Secure Key Generation

Cryptographic systems must ensure:

keys are generated randomly

weak keys are avoided

strong modern algorithms are used

3. DES is Deprecated

DES is now considered insecure due to:

small key size (56-bit)

known cryptographic weaknesses

Modern systems should use:

AES

ChaCha20
